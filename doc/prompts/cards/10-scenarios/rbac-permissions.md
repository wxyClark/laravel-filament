# 🔐 RBAC 权限场景模板

> **版本**: v3.0 | **层級**: L5+ | **最后更新**: 2026-06-07

## 场景概述
多层级权限系统的核心业务场景，包括角色管理、权限分配、资源级权限等。

## 领域模型扩展

### 表结构（Spatie Permission 包）
| 表名 | 说明 |
|------|------|
| `roles` | 角色 |
| `permissions` | 权限 |
| `model_has_roles` | 模型-角色关联 |
| `model_has_permissions` | 模型-权限关联 |

### 关键约束
- 高级角色自动继承低级角色权限
- 权限命名：`{resource}.{action}`
- 资源级权限使用 Gates
- 敏感操作 Gate 校验
