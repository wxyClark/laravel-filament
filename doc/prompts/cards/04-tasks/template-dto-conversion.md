# 任务模板：DTO 数据转换 (Data Transfer Object)

> **v2.0: 强制设计原理解释**

## 用途说明
规范从 HTTP Request 到业务逻辑层的数据传递方式，实现不可变性。

## 适用场景
- 处理复杂的表单提交数据。
- 在 Service 层之间传递标准化数据结构。
