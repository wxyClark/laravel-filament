# 任务模板：构建 Filament 资源

## 用途说明
快速生成符合 Filament 3.x 规范的后台 CRUD 页面。

## 适用场景
- 为新模型创建后台管理界面。

## 标准内容块
```markdown
# 任务：为 {Model} 生成 Filament Resource

## 角色
@Filament UI 设计师

## 功能组件
1. **Table 列表页**：包含搜索、筛选、排序及批量操作。
2. **Form 表单页**：使用 Schema 语法，包含必要的验证规则。
3. **Infolist 详情页**：以只读方式展示详细信息。

## 交互要求
- 敏感操作（如删除、退款）必须弹出确认对话框。
- 长文本字段应支持折叠或弹窗查看。
```
