# 任务模板：API Resource 响应格式化

> **v2.0: 强制设计原理解释**

## 用途说明
规范 API 响应的数据格式化，实现模型数据到 API 响应的标准化转换。

## 适用场景
- RESTful API 响应格式化
- 复杂的数据结构嵌套
- 条件字段暴露
