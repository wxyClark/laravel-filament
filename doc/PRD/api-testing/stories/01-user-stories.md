## Epic 1: 环境管理

### US-1.1: 创建测试环境

**作为** 测试人员
**我想要** 创建和管理测试环境配置
**以便** 在不同环境（开发/测试/生产）下执行接口测试

**验收标准**:
- [ ] 可以创建环境，填写名称、Base URL、认证方式
- [ ] 支持 JWT / Session / API Key / None 四种认证方式
- [ ] 认证配置（Token、API Key）加密存储
- [ ] 可以设置默认环境
- [ ] 可以复制已有环境配置

**优先级**: P0
**估算**: 5 故事点
