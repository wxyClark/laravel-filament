## 2. 仪表盘页面

### 2.1 统计卡片

| 卡片 | 数据 | 图标 | 颜色 |
|------|------|------|------|
| 总模块数 | `Module::count()` | folder | primary |
| 总接口数 | `Interface::count()` | cable | info |
| 总用例数 | `TestCase::count()` | document-duplicate | success |
| 今日执行数 | `TestExecution::whereDate('started_at', today())->count()` | play | warning |
| 今日通过率 | `(passed / total) * 100%` | check-circle | success/danger |

### 2.2 趋势图

**图表类型**: 折线图
**数据**: 最近 7 天的通过率趋势

```
X 轴: 日期 (MM-DD)
Y 轴: 通过率 (0-100%)
线条: 通过率曲线
标记: 数据点
```

**实现**: Filament Chart widget + Chart.js

### 2.3 最近执行记录

**显示字段**:
- 状态图标 (✅/❌)
- 执行 ID
- 通过率百分比
- 总耗时
- 执行时间

**交互**: 点击跳转到执行详情页

### 2.4 失败接口列表

**显示字段**:
- 接口名称
- 失败次数
- 最后失败时间
- 操作按钮 (查看/重试)

**排序**: 按失败次数降序

### 3.2 左侧接口树

**树节点类型**:

| 类型 | 图标 | 显示内容 |
|------|------|----------|
| 模块 | 📁 folder | 模块名称 + 接口数量 |
| 功能 | 📂 folder-open | 功能名称 + 接口数量 |
| 接口 | 🔌 cable | 接口名称 + 方法 + 最近状态 |

**最近状态图标**:
- ✅ `heroicon-o-check-circle` (绿色) - 最近测试通过
- ❌ `heroicon-o-x-circle` (红色) - 最近测试失败
- ⚪ `heroicon-o-question-circle` (灰色) - 未测试

**交互**:
- 点击节点：右侧显示详情
- 折叠/展开：点击箭头
- 搜索：实时过滤
- 拖拽排序：拖动节点调整顺序

### 3.3 右侧内容区域

**接口详情**:
- 接口名称
- HTTP 方法 + 路径
- 描述
- 操作按钮 (编辑/测试/删除)

**测试用例列表**:
- 用例名称
- 最近状态
- 最近执行时间
- 操作按钮 (测试/编辑/复制/删除)

## 5. 组件规范

### 5.1 统计卡片组件

```php
// Filament Widget
class StatsOverviewWidget extends Widget
{
    protected static string $view = 'filament.widgets.stats-overview';
    
    public function getStats(): array
    {
        return [
            Stat::make('总模块数', Module::count())
                ->description('接口模块')
                ->descriptionIcon('heroicon-o-folder')
                ->color('primary'),
            
            Stat::make('总接口数', Interface::count())
                ->description('API 端点')
                ->descriptionIcon('heroicon-o-cable')
                ->color('info'),
            
            Stat::make('总用例数', TestCase::count())
                ->description('测试用例')
                ->descriptionIcon('heroicon-o-document-duplicate')
                ->color('success'),
            
            Stat::make('今日通过率', $this->getTodayPassRate() . '%')
                ->description($this->getTodayExecutions() . ' 次执行')
                ->descriptionIcon('heroicon-o-check-circle')
                ->color($this->getTodayPassRate() >= 90 ? 'success' : 'danger'),
        ];
    }
}
```

### 5.2 接口树组件

```php
// Filament Page
class InterfaceTreePage extends Page
{
    protected static string $view = 'filament.pages.interface-tree';
    
    public function getTreeData(): array
    {
        return Module::with(['functions.interfaces.testCases.latestResult'])
            ->orderBy('sort_order')
            ->get()
            ->map(fn ($module) => $this->mapModuleToTreeNode($module))
            ->toArray();
    }
}
```

### 5.3 测试执行组件

```php
// Filament Action
class ExecuteTestAction extends Action
{
    public function handle(TestCase $testCase, Environment $environment): TestResult
    {
        return app(TestExecutorService::class)->execute(
            testCase: $testCase,
            environment: $environment
        );
    }
}
```

## 6. 交互规范

### 6.1 操作反馈

| 操作 | 反馈 |
|------|------|
| 执行测试 | Loading 动画 + 进度提示 |
| 测试通过 | 绿色 Toast 通知 |
| 测试失败 | 红色 Toast 通知 + 错误详情 |
| 保存成功 | 绿色 Toast 通知 |
| 删除确认 | Modal 确认弹窗 |

### 6.2 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Enter` | 执行测试 |
| `Ctrl + S` | 保存配置 |
| `Ctrl + F` | 搜索接口 |
| `Esc` | 关闭弹窗 |

### 6.3 响应式

| 断点 | 布局 |
|------|------|
| < 768px | 单列布局，树折叠 |
| 768px - 1024px | 双列布局 |
| > 1024px | 三列布局 |
