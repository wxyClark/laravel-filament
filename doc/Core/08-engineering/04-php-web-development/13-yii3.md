# 13. Yii3 实践要点

概要：针对 Yii3 的落地实践要点，结合常用模式与组件。

落地要点
- 路由与控制器：使用 REST 风格，控制器职责单一。
- 服务层解耦：通过接口与服务实现，便于单元测试和替换实现。
- 数据访问：Repository/ActiveRecord 的使用规范化，避免直接在控制器中编写 SQL。
- 安全与鉴权：CSRF、输入校验、密码哈希及会话管理。
- 测试：PHPUnit + Codeception 组合，覆盖控制器、服务与数据访问。

演示代码
```php
// Controller
class UserController extends Controller {
    private UserService $service;
    public function __construct(UserService $service) { $this->service = $service; }
    public function actionView(int $id) {
        return $this->asJson($this->service->getProfile($id));
    }
}

// Service
class UserService {
    private UserRepositoryInterface $repo;
    public function __construct(UserRepositoryInterface $repo) { $this->repo = $repo; }
    public function getProfile(int $id): ?array { return $this->repo->find($id); }
}
```
