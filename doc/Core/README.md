## 🎯 使用指南

### 如何使用这份文档

1. **外行人**：从感兴趣的学科开始，了解该学科的核心思维方式
2. **跨界者**：找到与自己专业相关的学科，学习可借鉴的方法论
3. **问题解决者**：遇到特定问题时，查找相关学科的解决方案

### 文档结构说明

每份文档包含：
- **常识误区**：该学科的人认为是常识，但外行人常误解的观点
- **核心方法**：该学科独有的、具有广泛用途的解决问题方法
- **跨界应用**：如何将该学科的方法应用到其他领域

## ✨ 新增内容亮点（v1.1）

### 🆕 数学思维方法论
**核心价值**：抽象化、逻辑推理、量化分析的通用工具

**核心方法**：
- **抽象化思维**：从具体问题提取模式，建立通用模型
- **证明思维**：严谨的逻辑推导和验证方法
- **建模思维**：用数学描述现实，预测和优化
- **概率思维**：量化不确定性，贝叶斯更新信念
- **优化思维**：在约束下寻找最优解

**跨界应用**：商业决策、时间管理、产品设计

---

### 🆕 心理学思维方法论
**核心价值**：理解人类行为、改善决策、促进行为改变

**核心方法**：
- **双系统思维**：快思考（直觉）vs 慢思考（理性）
- **认知偏差识别**：确认偏差、锚定效应、损失厌恶等
- **动机分析框架**：自我决定理论（自主、胜任、归属）
- **行为改变技术**：COM-B 模型、习惯养成、环境设计
- **情绪智力**：自我觉察、自我管理、社会觉察、关系管理

**跨界应用**：产品设计、团队管理、个人决策
